링크 태그
다른 페이지나 사이트로 연결 시키는 하이퍼링크(HyperLink)를 말합니다
<a href="이동할 파일명(절대경로,상대경로 둘다 가능)">클릭할 문자 또는 이미지 </a>
target 속성
<a> 태그가 연결된 문서를 어디에서 열지 설정합니다.
_blank :링크로 연결된 문서를 새창이나 새탭으로 열어줍니다
name값 : name으로 정의된 iframe을 통해 연결된 문서를 열어줍니다.

문제
index.html -> news.html 로 이름변경
index.html 새로 만들어 링크

홈으로(링크주소 index.html) | 인터넷 기사(링크주소 news.html) | 이력서(resume.html)
안녕하세요. 전승표의 웹사이트에 오신걸 환영합니다.

아이 프레임
아이프레임이란 inline frame의 약자입니다. iframe을 이용하면 웹 페이지 안에 또 다른 웹페이지를 삽입할 수 있습니다.
(비 표준 태그입니다. 표준으로 사용한다면 쓰지 않는것이 좋다)

<iframe src = "삽입할 페이지 파일 또는 링크" style="아이프레임의 크기를 설정할 css코드"></iframe>

HTML 요소의 타입

HTML 의 모든 요소는 해당 요소가 웹브라우저에 어떻게 보이는지를 결정짓는 display 속성을 가집니다.
대부분의 HTML 요소는 display 속성값으로 두가지 값 중 하나를 가지게 되는데, 블록과 인라인 속성입니다.

1.블록(block)
언제나 새로운 라인에서 시작되며, 해당 라인의 모든 너비를 차지합니다.
<p>,<h>,<ul>,<ol>,<div>,<form> 등

2.인라인(inline)
새로운 라인에서 시작하지 않습니다. 요소의 너비도 해당라인 전체가 아닌 HTML 요소의 내용만큼만 차지합니다.
<a>,<img>,<span> 등

<div>태그 
다른 HTML 요소들을 하나로 묶는데 자주 사용되는 대표적인 블록 요소입니다. 
<div> 요소는 주로 여러 요소들의 스타일을 한 번에 적용하기 위해 사용합니다.
주로 CSS에 사용됩니다.

<span> 태그
텍스트의 특정 부분을 묶는데 자주 사용되는 인라인 요소입니다. <span> 요소는 주로 텍스트의 
특정부분에 따로 스타일을 적용하기 위해 사용됩니다.

폼(form)태그
웹 페이지내에서 사용자로부터 입력을 받을 때 사용하는 태그입니다. 사용자가 입력한 데이터를 서버로 보낼때 form요소를 사용합니다

<form>
<form name="폼의 이름" method ="데이터 전송방식" action="서버측 데이터를 전달할 파일명">
입력양식... method방식 2가지 안보이게 전송하는 방식 혹은 눈에보이지만 빠르게 보내는방식(get post 방식)

</form>

<input> 태그
사용자로부터 데이터를 입력받기 위해 사용되는 태그입니다.
속성
type: 사용자로부터 어떠한 타입의 데이터를 입력받기 위함인지 설정합니다.
(text,password,radio,checkbox...)
<input> 태그의 type값
text: 사용자로부터 글을 입력받는 글상자입니다.
size: 글상자의 크기를 설정합니다.
*값의 길이가 아닙니다.
maxlength: 값의 최대길이를 설정합니다.
placeholder: 사용자가 입력 전에 특정 데이터를 입력하도록 안내해주는 값을 설정합니다.
value: 설정된 값을 서버로 전송하기 위해 사용합니다.
name: 요소의 이름을 설정합니다.(id,class)
readonly: 사용자가 입력 필드를 볼 수 있으나, 수정할 수 없게 설정합니다.
required: 사용자가 데이터를 필수로 입력 하도록 설정합니다.

<input> 태그의 type값
text: 사용자로부터 글을 입력받는 글상자입니다.
password: 사용자로부터 비밀번호를 입력받는 글상자입니다.
radio: 사용자로부터 여러개의 옵션중에서 단 하나의 옵션만을 입력 받을 수 있도록 사용하는 버튼입니다.
(단 서버로 정확한 입력을 전송하기 위해 반드시 name속성의 값이 값아야합니다.)
* checked 속성을 통해 여러 개의 옵션중에서 미리 선택되는 옵션을 지정할 수 있습니다.

///20200226 강의록 이어서...

checkbox: 사용자로부터 여러 개의 옵션 중에서 다수의 옵션을 입력 받을 수 있도록 사용하는 버튼입니다.
(단, 서버로 정확한 입력을 전송하기 위해 반드시 name 속성의 값이 같아야 합니다)

file : 사용자로부터 원하는 파일을 전송합니다. 
(단, 파일을 업로드시 form 태그에 반드시 enctype 속성을 작성해야 합니다)
예) enctype="multipart/form-data"

button : 아무런 이벤트가 없는 버튼(javascript와 연결하여 사용할 수 있습니다.)
submit : 사용자로부터 입력받은 데이터를 서버로 제출하는 버튼 입니다. 
(form 요소의 action속성에 명시한 파일로 데이터를 제출합니다.)
reset : 사용자로부터 입력받은 데이터를 초기화하는 버튼입니다.
color:  사용자로부터 원하는 색상을 선택하여 입력하게 하는 버튼입니다.
16진수 -> 0 ~ f 까지의 수
16진수 RGB컬러로 색상을 표현
예) #000000 : 검정색
    -- -- --
    R  G  B
    #ffffff : 흰색
    #ff0000 : 빨간색
    #00ff00 : 초록색
    #0000ff : 파랑색

email : 사용자로부터 이메일을 형식에 맞게 입력 받습니다. 
(단, '@' 빠지면 입력란 하단에 풍선도움말로 입력 오류를 알려줍니다. 하지만 강제성은 없습니다)

url : 사용자로부터 웹 사이트를 형식에 맞게 입력 받습니다.
(단, 'http:// '빠지면 입력란 하단에 풍선 도움마로 입력 오류를 알려줍니다.
 하지만 강제성은 없습니다.)

tel : 사용자로부터 전화번호를 입력 받습니다. 유효성 체크를 하지 않으며 (왜냐하면 전세계 번호형식은 다 다르니깐..)
자동으로 '-'을 넣어주지 않습니다. 
(단, 모바일 기기에서 입력할 때 자동으로 숫자패드가 나타납니다. 하지만 pc에서 입력시
 문자가 입력되어도 강제성은 없습니다.)

date : 사용자로부터 원하는 날짜를 입력 받을 수 있습니다.

number : 사용자로부터 원하는 숫자를 입력 받을 수 있습니다.
<input type = "number" min="최소값" max="최대값" step="증가값">

<select>

<select name="select 이름" multiple="선택할 갯수">
<option value="값1"> 텍스트1</option>
<option value="값2"> 텍스트2</option>
<option value="값3"> 텍스트3</option>

<textarea> 
사용자로부터 여러줄의 글을 입력 받을 수 있는 글상자입니다.

<textarea cols="한 라인에 입력받을 글자수" rows="보여줄 라인수">값</textarea>
